# 01 TypeScript 安装

# 安装

基于node.js安装（前提安装node.js）

npm install -g typescript

# 编译

新建一个helloworld.ts文件，同级目录下命令行执行：

tsc helloworld.ts

完成后就会在同级目录生成同名JS文件helloworld.js。

开发项目级应用每次都tsc进行编译，显然很不方便。下面就需要对项目进行简单配置。

# 第一个TypeScript应用-Helloworld

创建一个helloworld.html文件，输入html:5回车 快捷生成一个简单页面。

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Hello world</title>
</head>
<body>
    
</body>
</html>

引入.js文件：

<script src="helloworld.js" type="text/javascript"></script>

helloworld.ts

console.log('hello TypeScript!');

var hello:string = 'hello 你好';

document.writeln(hello);

即可看到输出结果,生成对应js文件:

"use strict";
console.log('hello TypeScript!');
var hello = 'hello 你好';
document.writeln(hello);

修改.ts文件，页面也即是响应更新。
