# Typescript 介绍

官网简介

TypeScript is a typed superset of JavaScript that compiles to plain JavaScript.

Any browser. Any host. Any OS. Open source.

TypeScript是一个类型化的JavaScript超集，可以编译成普通JavaScript。

任何浏览器。任何东道主。任何操作系统。开源。

  1. TypeScript是由微软开发的一款开源编程语言。
  2. TypeScript是JavaScript的超级，遵循最新ES6、ES5规范。TypeScript扩展了JavaScript的语法。
  3. TypeScript更像后端面向对象语言可以让js开发大型企业级项目应用。
  4. 谷歌Angular2.x+就是基于TypeScript语法。
  5. 最新的Vue、React也可以集成TypeScript。
